== Description ==

Wp-xPerts woocommerce custom thank you page plugin enables you to create a custom thank you page.
Admin will be redirected to custom thank you page after successful checkout.
This way you can create a custom thank you message too

Options

* select target redirect page
* option for custom/external URL added


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Woo Thank-You Page to manage redirect.
